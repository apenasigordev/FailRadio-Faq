# FailRadio
 - Failradio é um bot para Discord grátis e fácil de usar
 - Com um simples sistema de estações para você escutar músicas diretamente **DO YOUTUBE**

# Como usar
 - Prefixo atual: f!
Para você escutar estações use f!radio *Numero da estação*
